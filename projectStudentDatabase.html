Student Database

Store Student Records

The Brief

This application will get data from a database of students and it will list all the students in a data grid, the user can click on of the items an see all the information in that student.

The user can also add, delete or update records in the database, all the text inputs are validated to prevent the user from entering incorrect data.

The application also has an about and instructions page to help the user undertand the apllication.

What I Did

The application was designed and coded by me using visual studio and the c# language, the program was deployed to a .exe file for testing purposes
