Vending Machine

Lets users set up and maintain vending machines

The Brief

This application allows a user to create, load, and service (test functionality of) vending machines. The user is also able to adjust prices of products and generate a sales report showing the sales totals of each item.

The bending machine records are stored in a .txt file, files can be loaded with the information of the machine already in it.

The application also lets the user add or delete items in the vending machines

I coded the file read and write of the program, I also helped the team to integrate all the coding parts toghether.

This project helped me to get experience working with a team.
